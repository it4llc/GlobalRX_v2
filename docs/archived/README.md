# Archived Documentation

This directory contains documentation that has been superseded or is no longer actively maintained but is kept for historical reference.

## Directory Structure

### `/mvp-planning/`
Contains initial MVP planning documents that were used during the planning phase but have been superseded by the actual implementation documents:

- `mvp-status-audit.md` - Original spec for the MVP audit (superseded by `/docs/MVP_STATUS_AUDIT.md`)

## Why Archive?

These documents are archived rather than deleted to:
1. Preserve project history
2. Allow comparison between planned vs actual implementation
3. Provide context for future development decisions
4. Maintain audit trail of project evolution

## Current Active MVP Documentation

For current MVP status and documentation, please refer to:
- `/docs/MVP_COMPLETE.md` - MVP achievement milestone document
- `/docs/MVP_STATUS_AUDIT.md` - Complete feature inventory and status
- `/docs/MVP_IMPLEMENTATION_PLAN.md` - Implementation roadmap and results
